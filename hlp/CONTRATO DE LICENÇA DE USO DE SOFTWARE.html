Termos usados neste contrato:

Drivers : São pequenos programas instalado no computador pelo sistema operacional ou por programas terceirizados necessários para o funcionamento de um componente especifico do computador, como por exemplo o driver de teclado, que é necessário para o funcionamento do teclado.

Executável de instalação : É um programa que pode ser executado em um computador e instala nele um determinado conjunto de programas, estes programas geralmente são obtidos através de um CD de instalação ou através da Internet. Por exemplo, o Programa Vias Urbanas & Estradas é formado por um conjunto de subprogramas. Este conjunto de subprogramas é acondicionado em um único pacote que pode ser aberto pelo licenciado. Este pacote é o executável de instalação. Sua abertura desencadeia uma série de procedimentos computacionais que instala o programa Vias Urbanas & Estradas no computador do licenciado. Um destes procedimentos é a apresentação deste contrato ao licenciado questionando-o se ele concorda plenamente ou não com os termos nele contidos. A instalação só é feita caso o licenciado concorde.

Hard Lock: É um componente físico que pode ser conectado ao computador pelo licenciado e contém chaves de segurança. Estas chaves de segurança se válidas permitem ao licenciado que use o programa. O programa Vias Urbanas & Estradas é fornecido com um hard lock na forma de pendrive. Para que o programa Vias Urbanas & Estradas funcione é necessário que este pendrive esteja conectado ao computador.

Hardware: São componentes físicos ligados eletronicamente ao computador, como por exemplo, o teclado ou o monitor de vídeo. As placas ligadas internamente também são hardware, como por exemplo a placa de vídeo.

Não Exclusiva : Significa que estas licença não é exclusiva do licenciado. A Boole Software poderá fornecê-la a outras pessoas, físicas ou jurídicas.

Não Redistribuível :  Significa que o licenciado não tem o direito de redistribuir este programa a outras pessoas, físicas ou jurídicas. Este direito é exclusivo da Boole Software Ltda.

Não Sublicenciável :  Significa que o licenciado não tem o direito de licenciar este programa a outras pessoas, físicas ou jurídicas. Este direito é exclusivo da Boole Software Ltda.

Pacotes Redistribuíveis (redistributable packs) :  São conjuntos de programas instalados no computador necessários ao funcionamento de outros programas que foram desenvolvidos usando-os como base. Por exemplo, O programa Vias Urbanas & Estradas foi desenvolvido usando-se o ambiente de programação chamado Visual Studio da Microsoft, portanto, para que o programa Vias Urbanas & Estradas funcione  é necessário que o Service Pack referente ao Visual Studio esteja instalado no computador no qual se pretende usar o Vias Urbanas & Estradas. Pacotes de Serviços (Serviçes packs) :  São conjuntos de programas instalados no computador necessários ao funcionamento de um determinado programa ou do sistema operacional, geralmente instalados para corrigir erros das versões anteriores dos programas. Estes pacotes não podem ser redistribuídos pelo usuário.
